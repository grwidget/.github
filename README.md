GR Widget
Live Google review embeds for websites — no API keys for site owners.

What we build
GR Widget connects to Google Business Profile listings and serves customisable review widgets via a lightweight JavaScript embed. Works on:

WordPress (official plugin + shortcode/block)
Static HTML / Jamstack sites
Wix, Webflow, Shopify (HTML embed)
Any CMS with a custom HTML block
Features
13 widget layouts (badge, carousel, grid, list, corner sticker, flyout)
Automatic review sync
Colour and typography customisation
Agency multi-widget dashboard
JSON-LD on eligible paid plans
Quick start
1. Register at https://grwidget.com/register
2. Search your Google Business Profile
3. Copy embed code
4. Paste into your site
